# Sistema de Busca Otimizado para Catálogo de Produtos - MegaStore

## 📌 Descrição

Este projeto foi desenvolvido como parte da disciplina **Data Structure Strategy and Implementation**.

O objetivo é implementar um sistema de busca eficiente e escalável para um catálogo de produtos, utilizando a linguagem Rust e estruturas de dados otimizadas.

---

## 🚀 Funcionalidades

- Indexação eficiente de produtos
- Busca rápida por nome, marca e categoria
- Sistema de recomendação com grafos
- Recomendações indiretas utilizando BFS
- Medição de desempenho
- Testes automatizados

---

## 🛠 Tecnologias Utilizadas

- Rust
- HashMap
- HashSet
- VecDeque
- Cargo
- Cargo Test
